# 地下謎への招待状 2025

地下謎への招待状 2025 特設サイトです。

## 作業を始めるにあたって

ejs, sass, es6 を gulp でコンパイルしています。リポジトリをクローンしたら、以下の手順で開発環境を構築してください。

### パッケージのインストール

```
yarn install
```

### 開発環境（各ファイルの watch）

```
yarn run dev
yarn gulp dev
```

### プロダクション環境（静的ファイルの生成）

```
yarn run build
yarn gulp build
```
